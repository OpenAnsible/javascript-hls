JavaScript HTTP Live Streaming
=====================================

:Date: 12/18 2016


.. contents::

介绍
-----

基于 `HTML5` 新增特性 `Media Source Extensions <https://w3c.github.io/media-source/>`_ 实现的音视频媒体播放器。
